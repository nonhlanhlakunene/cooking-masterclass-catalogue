# Cooking Masterclass Catalogue

**Trainee:** Nonhlanhla Kunene
**Programme:** YouthCode On-Site — Cohort 18, 2026
**Course:** Course 1 — Frontend Web Development

## Project Overview
A responsive Vue.js single-page catalogue application for Cooking Masterclass — an online platform curating cooking workshops hosted by expert chefs. Users can browse available courses, view pricing and skill levels, and save courses to a wishlist.

## Features
- Course listings displayed as cards
- Wishlist functionality with counter in the header
- Sold Out indication for unavailable courses
- Responsive layout for desktop and mobile

## Tech Stack
- Vue 3 (Vite)
- JavaScript
- HTML5
- CSS3

## Installation and Run Instructions
1. Clone the repository
2. Navigate into the project folder
3. Run `npm install`
4. Run `npm run dev`
5. Open your browser and go to the local address shown in the terminal
